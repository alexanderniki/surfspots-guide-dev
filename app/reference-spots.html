Типы спотов
Тип спота зависит от места, на котором образуются волны. Например:
  • Beach break
  • Reef break
  • Point break
  • Не известно
Направление волны
Направление, в котором ломается большая часть волн на споте. Например:
  • Левая
  • Правая
  • Не известно
Дно
Основное покрытие дна. Например:
  • Песок
  • Галька
  • Камни
  • Не известно
Возможны другие варианты по ситуации.
Сложность
Субъективная сложность катания на споте. Например:
  • Начинающий
  • Продвинутый
  • Эксперт
Уровень загруженности
Показатель того, насколько спот популярен и насколько на нем людно. Например:
  • Низкий
  • Средний
  • Высокий
Угрозы
Проблемы, с которыми можно столкнуться на споте. Например:
  • Камни
  • Течение
  • Нет
Возможны другие варианты по ситуации.
Качество волн
Качество волн по местным локальным меркам. Например:
  • Низкое
  • Среднее
  • Высокое
