SURFL О проекте

SURFL - это коллекция структурированной систематизированной практической информации о сефр-спотах для людей, которые уже знают о серфинге и хотят знать, где можно покататься и какие специфические нюансы для конкретной локации существуют.

Цель проекта:

  • собрать в одном месте стуктурированную систематизарованную информацию спотах, правилах поведения на них
  • предоставить людям информацию о специфических нюансах серфинга в выбранной локации (правила, лайфхаки, рекомендации по экипировке)

Лицензия

Информация для проекта взята из открытых источников, личного опыта и переписки с участниками серф-сообщества.

Если явно не указано иное, то вся авторская информация (тексты, фотографии, иллюстрации и пр.) рапространяется по лицензии Creative Commons Attribution Share Alike 4.0 (CC BY-SA 4.0) и является открытой и общедоступной.

Почитайте про Creative Commons и OpenSource-сообщество [1] [2], если хотите лучше понимать, почему это важно.

Чем проект не является

SURFL не является энциклопедией серфинга.

Вы не найдете здесь статей или уроков по теории серфинга.

О теории серфинга, серф-культуре, соревнованиях, досках и пр. пишет множество других ресурсов.

Используемые технологии

В работе над SURFL используется множество открытых продуктов и инструментов, которые заслуживают упоминания.

  • Шрифт логотипа: Orbitron (Github)
  • Основной шрифт для контента: PT Root UI от студии Paratype
  • Иконки: materialdesignicons.com
  • Среда разработки: Visual Studio Code
  • Система контроля версий: Git
  • API погоды: OpenMeteo
